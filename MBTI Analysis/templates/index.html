MBTI Analysis of Youtube Video Comments

About This Project

Goal
The goal is to determine personality composition of the comments of any given YouTube video.
How it Works
1. The User provides a Youtube URL.
2. The Server checks the validity of the link and informs the User.
3. The Server scrapes all comments associated with the YouTube video.
4. The comments are analyzed using the propietary NLP AI for their MBTI characteristics.
5. The data is displayed to the user in the Output section.

Input &Output
Input: YouTube video URL must be copied and pasted into the textbox in the Input section, then click Enter.
Output:Raw Data file available for download along with a couple of graphs displayed in the Output Section.
